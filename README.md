# payi-utilities

Utility repositories and tooling for Pay-i integrations.
This is a new toolkit for customers who use N8N for faircode IT operations. See Docs for more Details.

## Repositories in this workspace

- `n8n-toolkit/`: migration and audit toolkit for n8n workflows routed through Pay-i.

## Sync model

`n8n-toolkit/` is imported from its own repository using `git subtree`.
